# El Futuro de las Promesas

[Built with revealjs](http://revealjs.com/).

# Quickstart

You need [Node.js](http://nodejs.org/) (10+ or later).

```sh
# 1. Clone the repository
git clone https://github.com/camolin3/promises-talk-2018.git

# Navigate to the promises-talk-2018 folder
cd promises-talk-2018

# Install dependencies
npm ci

# Serve the presentation and monitor source files for changes
npm start
```

Open <http://localhost:8000> to view the presentation.

Press `s` to open speaker view with notes.

Press `f` to go full screen.
